1] delete a branch : 
    
    git branch -d branch_name // لو تعديلات ملهاش كوميت مش هيرضي يمسح

    git branch -D branch_name // لو في تعديلات مالهاش كوميت هيمسحها بالقوه


2] Modify a commit 

    git commit --amend but before it write 
    git config --global core.editor "code --wait" // for more useful GUI 

3] How to create an alias

     git config --global alias.cm "commit -m"

     git config --global alias.st status

     git config --global alias.br branch
 
4] How to create a new branch and go to it.

    git checkout -b branch_name

5] How to rename an exiting branch.

    git branch -m new_branch_name
