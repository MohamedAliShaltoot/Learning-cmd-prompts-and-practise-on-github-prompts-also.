1] delete a branch : 
    
    git branch -d branch_name // لو تعديلات ملهاش كوميت مش هيرضي يمسح

    git branch -D branch_name // لو في تعديلات مالهاش كوميت هيمسحها بالقوه


2] Modify a commit 

    git commit --amend but before it write 
    git config --global core.editor "code --wait" // for more useful GUI 

3] How to create an alias

     git config --global alias.cm "commit -m"

     git config --global alias.st status

     git config --global alias.br branch
 
4] How to create a new branch and go to it.

    git checkout -b branch_name

5] How to rename an exiting branch.

    git branch -m new_branch_name

6] How to disappear specific files.

    git stash save " msg "
    هنا لو عملت تعديلات بس مش عاوز ارفعها دلوقت وعاوز اكمل بكتب الكومند دا 

    git stash pop  هنا بقا برجع الملفات تاني اقدر اعملها كوميت وارفعها ولكن بيشيلها من هناك

     git stash apply  هنا بيجيب اخر ملف لسه متضاف بس بيسيب نسخه منه هناك

     git stash list هنا بيعرض كل الملفات بال index بتاعها 

     git stash drop stash@{id} --> to delete specific file

     git stash show  --> to show what is in stash list



    ----
    If we want a specific file from stashed list?

         git stash pop stash@{id}
